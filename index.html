My Cheat Sheet

CSS link

<link href="styles.css" rel="stylesheet">

De href moet het pad naar het css bestand zijn.

Background Image

url(https://www.image.jpg/);

Je krijgt een leuke achtergrond.

Span

<p><span>different style</span>the rest of the paragraph<p>

Span wordt gebruikt om in een paragraaf een stukje anders te stylen.

Gradient

linear-gradient(180deg, red, yellow);

Maakt een lineare gradient tussen 2 kleuren op een bepaalde hoek.

Hover

.class:hover{}

Geeft een andere style voor mouse hover.

Font-family

font-family: Helvetica, sans-serif;

Het tweede en volgende fonts zijn de backup fonts voor als de fonts ervoor niet beschikbaar zijn.

Border-box

box-sizing: border-box;

Zorgt ervoor dat de margin en border inbegrepen zijn bij de width en height.

Z-index

z-index: 1;

Wanneer er meerdere elementen elkaar overlappen, ligt de hoogste z-index element bovenop.
